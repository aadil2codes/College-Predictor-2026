# 🎓 JEE College Predictor & AI Counseling Assistant

A smart AI-powered platform that helps JEE Main aspirants discover colleges based on previous year JoSAA and CSAB cutoff data.

The platform combines traditional cutoff prediction with an intelligent AI counseling assistant to provide a more realistic and interactive college selection experience.

---

🔗Live Preview: https://college-predictor-2026.vercel.app/

## 🚀 Features

### 🎯 College Prediction

* Rank-based college prediction
* JoSAA cutoff support
* CSAB cutoff support
* NIT predictions
* IIIT predictions
* GFTI predictions
* Branch-wise filtering

### 📊 Smart Filtering

* Category filter
* Quota filter
* Gender filter
* Counseling type selection
* Round-wise prediction

### 🟢 Safe & 🟡 Low Chance Prediction

Unlike traditional predictors, this platform also shows colleges that are slightly beyond the previous year's cutoff using realistic cutoff-shift analysis.

* 🟢 Safe Colleges
* 🟡 Low Chance Colleges
* Dynamic rank-buffer system
* Realistic cutoff trend analysis

### 🤖 AI Counseling Assistant

* Natural conversation support
* Gemini-powered chatbot
* Smart counseling flow
* Context-aware responses
* College recommendation guidance
* Dataset-based prediction responses

### 📱 Progressive Web App (PWA)

* Installable on Android
* Add to Home Screen support
* Mobile-friendly design
* App-like experience
* Responsive UI

2. Select category, quota, gender, and counseling type
3. View Safe and Low Chance colleges
4. Use the AI assistant for personalized guidance
5. Explore NIT, IIIT, and GFTI options
